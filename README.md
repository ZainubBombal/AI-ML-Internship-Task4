# Task 4: General Health Query Chatbot

## Objective

Develop a health-related chatbot using a Large Language Model (LLM) and prompt engineering techniques.

## Dataset

Healthcare Patient-Doctor Conversation Dataset (50K Samples)

## Technologies Used

* Python
* Hugging Face Transformers
* Pandas
* Google Colab

## Features

* Prompt Engineering
* Friendly Medical Assistant Behavior
* Safety Filters
* Interactive Chat Interface

## Example Queries

* What causes a sore throat?
* Is paracetamol safe for children?

## Safety Measures

The chatbot avoids providing harmful medical advice and encourages consultation with healthcare professionals.

## Conclusion

A general health query chatbot was developed using a pre-trained language model with prompt engineering and safety mechanisms.
